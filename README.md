Convert Jupyter Notebooks into organized Python packages.

## Live Application

Try it now: [https://notebook-converter.streamlit.app/](https://notebook-converter.streamlit.app/)

## Features

- **Batch Processing**: Convert multiple `.ipynb` files simultaneously
- **Code Extraction**: Extract code into clean Python scripts
- **Output Preservation**: Save cell execution outputs in structured text files
- **Full Documentation**: Generate markdown files that preserve the notebook structure with code blocks, outputs, and original markdown cells
- **Image Extraction**: Extract images from cell outputs (PNG, JPEG, GIF)
- **Statistics**: View detailed metrics about processed notebooks
- **Advanced Options**: Remove magic commands, add cell numbers, custom naming, encoding selection

## Output Structure

For each processed notebook, you receive:

```
notebook_name/
├── notebook_name.py               # Python code from all code cells
├── notebook_name_outputs.txt      # Cell execution outputs
├── notebook_name_documentation.md # Complete notebook as markdown
├── notebook_name_image_1.png      # Extracted images
└── notebook_name_image_2.png
```

### Documentation Format

The generated markdown file preserves the notebook structure:

```markdown
# notebook_name

> Extracted from `notebook_name.ipynb`

---

[Original markdown cells appear here]

### Code Cell 1

```python
import pandas as pd
df = pd.read_csv('data.csv')
```

**Output:**

```
   col1  col2
0     1     2
1     3     4
```

[More cells...]

---

*Generated by Notebook Converter | 5 code cells, 3 markdown cells, 2 images*
```

## Installation

### Prerequisites

- Python 3.10 or higher
- pip

### Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/JuanLara18/notebook-converter.git
   cd notebook-converter
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   streamlit run app.py
   ```

### Development Installation

```bash
# Using Poetry
poetry install

# Or pip with dev dependencies
pip install -e ".[dev]"
```

## Usage

1. Upload one or more `.ipynb` files
2. Configure export options in the sidebar
3. Click "Process Files"
4. Preview the extracted content
5. Download the ZIP package

## Configuration Options

| Option | Description | Default |
|--------|-------------|---------|
| Include outputs | Export cell execution outputs | Yes |
| Include images | Extract images from outputs | Yes |
| Include markdown | Generate documentation file | Yes |
| Remove magic commands | Strip IPython `%` and `!` commands | No |
| Add cell numbers | Add `# --- Cell N ---` comments | No |
| Custom ZIP name | Customize output filename | Auto |
| Encoding | Output file encoding | UTF-8 |

## Notes

- All files are processed in-memory
- IPython magic commands can be optionally removed
- HTML and JavaScript outputs are not processed
- Large notebooks may take longer to process
