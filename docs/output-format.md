# Output Format

## ZIP Structure

For each processed notebook, you receive a folder inside the ZIP:

```
notebook_name/
├── notebook_name.py               # Python code from all code cells
├── notebook_name_outputs.txt      # Cell execution outputs
├── notebook_name_documentation.md # Complete notebook as markdown
├── notebook_name_image_1.png      # Extracted images
└── notebook_name_image_2.png
```

## Documentation Format

The generated markdown file preserves the full notebook structure — original markdown cells, code with syntax highlighting, and cell outputs inline.

Example:

~~~markdown
# notebook_name

> Extracted from `notebook_name.ipynb`

---

[Original markdown cells appear here]

### Code Cell 1

```python
import pandas as pd
df = pd.read_csv('data.csv')
```

**Output:**

```
   col1  col2
0     1     2
1     3     4
```

[More cells...]

---

*Generated by Notebook Converter | 5 code cells, 3 markdown cells, 2 images*
~~~
